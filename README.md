# RSS Telegram Feed

自动生成 Telegram 频道 [@Githubrebang](https://t.me/Githubrebang) 的 RSS 订阅源。

由 GitHub Actions 每 30 分钟自动更新，托管在 GitHub Pages 上，24 小时免费运行。

## 订阅地址

```
https://birdplus.github.io/rss-telegram-feed/feed.xml
```

## 技术栈

- Python 3 自动抓取 Telegram 公开页面
- GitHub Actions 定时执行（每 30 分钟）
- GitHub Pages 托管 RSS 文件
